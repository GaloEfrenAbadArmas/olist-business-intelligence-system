## Descripción del proyecto

Este repositorio contiene el desarrollo técnico del Trabajo Fin de Máster orientado al diseño e implementación de un sistema de inteligencia comercial basado en datos reales de comercio electrónico.

El proyecto utiliza el dataset público de Olist para construir una solución analítica completa que integra procesos de preparación de datos, análisis exploratorio, modelado dimensional para Business Intelligence y técnicas de aprendizaje automático para la segmentación de categorías de producto.

---

## Objetivos

- Integrar y transformar datos procedentes de múltiples entidades de negocio.
- Desarrollar una capa analítica orientada a la toma de decisiones.
- Construir un modelo dimensional para su explotación en Power BI.
- Diseñar indicadores clave de rendimiento (KPIs).
- Analizar el comportamiento comercial, logístico y geográfico del marketplace.
- Aplicar técnicas de clustering para segmentar categorías de producto.

---

## Metodología

El proyecto se desarrolló siguiendo las siguientes fases:

1. Comprensión y revisión inicial de los datos.
2. Preparación y transformación de datos.
3. Ingeniería de características (Feature Engineering).
4. Análisis Exploratorio de Datos (EDA).
5. Construcción del dataset para Power BI.
6. Preparación de datos para modelos analíticos.
7. Desarrollo y evaluación de modelos.
8. Aplicación del algoritmo K-Means.
9. Diseño del dashboard de Business Intelligence.

---

## Principales resultados

El análisis permitió identificar patrones comerciales, logísticos y geográficos relevantes para el negocio. Asimismo, la aplicación del algoritmo K-Means permitió segmentar las categorías de producto en cuatro grupos diferenciados:

- Categorías core.
- Categorías premium.
- Categorías estándar.
- Categorías logísticamente ineficientes.

Estos resultados facilitaron la generación de recomendaciones orientadas a mejorar la rentabilidad y la eficiencia operativa del marketplace.
